
To use the following files you must export the **GNOME15_GIT** variable
in your environment. Example :

```
$ grep GNOME15 ~/.bashrc
export GNOME15_GIT="/path/to/my/gnome15/git/repository"
```

1. Makefile

2. vim.project
--------------
